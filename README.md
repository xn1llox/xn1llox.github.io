# Espinoza Informática · IT Solutions

Sitio web profesional de **Informática Felipe Andrés Espinoza E.I.R.L.**, empresa de soluciones tecnológicas con base en Parral, Región del Maule, Chile.

🌐 **Demo en vivo:** [https://espinozainformatica.cl/](https://espinozainformatica.cl/)

---

## Descripción

Sitio web estático de una sola página (`index.html`) con diseño dark tech moderno, desarrollado para presentar los servicios de soporte técnico, desarrollo de software y soluciones IT para personas, empresas y pymes.

## Características

- Diseño dark tech con paleta de colores azul/cyan sobre fondo oscuro
- Navbar fija con logo de empresa y scroll suave entre secciones
- Hero con tarjeta estilo terminal animada
- Sección **Sobre Nosotros** con foto del representante y perfil profesional
- Sección **Servicios** con cards con efecto glow al hover
- Sección **Trabajos Realizados** con galería y lightbox integrado
- Sección **Contacto** con datos, redes sociales y botón WhatsApp
- Botón flotante de WhatsApp
- Animaciones de entrada con AOS (Animate On Scroll)
- 100% responsive (mobile-first con Bootstrap 5)
- Listo para GitHub Pages (sin dependencias de servidor)

## Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 + CSS3 | Estructura y estilos personalizados |
| [Bootstrap 5.3](https://getbootstrap.com/) | Layout responsive y componentes |
| [Bootstrap Icons 1.11](https://icons.getbootstrap.com/) | Iconografía |
| [AOS 2.3](https://michalsnik.github.io/aos/) | Animaciones al scroll |

## Despliegue en GitHub Pages

1. Crear repositorio con nombre `xn1llox.github.io`
2. Subir `index.html` y la carpeta `img/`
3. Ir a **Settings → Pages → Branch: main → Save**
4. El sitio estará disponible en `https://xn1llox.github.io`
